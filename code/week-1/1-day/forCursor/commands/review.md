# Команда: /review

## Описание
Проведи детальное code review выбранного кода или файла.

## Prompt
Проанализируй этот код и проверь:

1. **Безопасность:**
   - SQL injection, XSS, CSRF уязвимости
   - Валидация пользовательских данных
   - Хранение секретов и паролей

2. **Производительность:**
   - Неоптимальные циклы и алгоритмы
   - N+1 проблемы в запросах
   - Лишние ре-рендеры (для React)
   - Memory leaks

3. **Читаемость:**
   - Понятные имена переменных/функций
   - Соответствие Single Responsibility Principle
   - Избыточная сложность

4. **Обработка ошибок:**
   - Try-catch для критичных операций
   - Валидация входных данных
   - Логирование ошибок

5. **TypeScript:**
   - Отсутствие `any`
   - Правильная типизация
   - Type guards где необходимо

**Результат:**
Выдай список найденных проблем с приоритетом (🔴 критично, 🟡 важно, 🟢 рекомендация) и примеры исправлений.
