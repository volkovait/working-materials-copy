# Команда: /refactor

## Описание
Рефактори код для улучшения читаемости и поддерживаемости.

## Prompt
Отрефактори этот код согласно best practices:

1. **Clean Code принципы:**
   - **Single Responsibility:** Каждая функция делает одно
   - **DRY:** Устрани дублирование кода
   - **Понятные имена:** Переименуй неясные переменные/функции
   - **Магические числа:** Замени на именованные константы

2. **Разделение логики:**
   - Выдели бизнес-логику в отдельные функции
   - Отделяй data fetching от rendering (React)
   - Создавай utility функции для повторяющихся операций

3. **TypeScript улучшения:**
   - Замени `any` на конкретные типы
   - Создай interfaces для объектов
   - Используй union types для перечислений
   - Добавь type guards

4. **Паттерны проектирования:**
   - Используй Strategy для различных вариантов поведения
   - Factory для создания объектов
   - Dependency Injection для тестируемости

5. **Структура:**
   - Группируй связанный код
   - Выноси константы в начало файла
   - Сортируй импорты логически

**Результат:**
Отрефакторенный код с объяснением каждого изменения и почему это улучшение.
