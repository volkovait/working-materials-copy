# Команда: /debug

## Описание
Найди и исправь баг в коде.

## Prompt
Помоги найти и исправить баг:

**Проблема:** [опиши проблему или вставь error message]

**Анализ:**

1. **Root Cause Analysis:**
   - Изучи stack trace
   - Определи точное место ошибки
   - Найди причину (не симптом, а корень проблемы)

2. **Проверь типичные ошибки:**
   - Off-by-one errors в циклах
   - Null/undefined доступы
   - Асинхронные race conditions
   - Memory leaks (неочищенные listeners, intervals)
   - Неправильная обработка Promise
   - Мутация immutable данных (React state)

3. **Воспроизведение:**
   - При каких входных данных происходит ошибка?
   - Граничные условия (empty, null, edge cases)
   - Последовательность действий для воспроизведения

4. **Решение:**
   - Предложи исправление с объяснением
   - Добавь проверки для предотвращения подобных багов
   - Предложи тест, который бы поймал этот баг

5. **Профилактика:**
   - Как избежать подобных проблем в будущем?
   - Нужны ли дополнительные проверки?
   - Стоит ли изменить архитектуру?

**Результат:**
- Объяснение причины бага
- Исправленный код с комментариями
- Тест для проверки fix
- Рекомендации для предотвращения
