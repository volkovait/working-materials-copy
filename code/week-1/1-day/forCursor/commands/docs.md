# Команда: /docs

## Описание
Создай полную документацию для кода.

## Prompt
Создай документацию для этого кода:

1. **JSDoc/PyDoc комментарии:**
   ```typescript
   /**
    * Краткое описание функции
    * 
    * @param paramName - Описание параметра
    * @returns Описание возвращаемого значения
    * @throws {ErrorType} Когда возникает ошибка
    * @example
    * const result = functionName(param);
    * console.log(result); // Ожидаемый результат
    */
   ```

2. **API документация (для endpoints):**
   - HTTP метод и путь
   - Описание функциональности
   - Request body схема с примером
   - Response схема с примерами (success и error)
   - Коды ответов и их значения
   - Требования авторизации

3. **README секция (для модулей):**
   - Краткое описание назначения
   - Примеры использования
   - Список зависимостей
   - Конфигурация (если есть)

4. **Inline комментарии:**
   - Объяснение сложной бизнес-логики
   - Почему выбрано именно это решение
   - Known issues или workarounds

Используй четкие формулировки и конкретные примеры.
