# Команда: /optimize

## Описание
Оптимизируй код для лучшей производительности.

## Prompt
Проанализируй этот код и оптимизируй его:

1. **Алгоритмическая сложность:**
   - Проверь временную сложность (O(n), O(n²), etc.)
   - Найди возможности уменьшить сложность
   - Предложи более эффективные алгоритмы

2. **Структуры данных:**
   - Используй оптимальные структуры (Map вместо Object, Set вместо Array для уникальных значений)
   - Избегай лишних копирований данных

3. **React оптимизации (если применимо):**
   - Добавь useMemo для дорогих вычислений
   - Добавь useCallback для функций в dependencies
   - Используй React.memo для компонентов
   - Избегай лишних ре-рендеров

4. **Database запросы (если применимо):**
   - Добавь индексы для часто используемых полей
   - Используй batch операции вместо множества одиночных
   - Избегай N+1 проблемы
   - Используй SELECT только нужных полей

5. **Общие оптимизации:**
   - Lazy loading где возможно
   - Кэширование результатов
   - Debounce/throttle для частых операций
   - Virtual scrolling для больших списков

**Результат:**
Покажи оптимизированный код с объяснением изменений и замерами производительности (до/после).
