# 📚 Примеры использования промптов

## 1. ✂️ Сжатие текста

Сжатие текста — одна из базовых, но крайне полезных задач. Ниже показано, как с помощью промпта получать короткие резюме.

Предположим, вы хотите узнать об антибиотиках.

Промпт:
```
Объясни, что такое антибиотики.
Ответ:
```

Результат:
```
Антибиотики — это тип лекарственных препаратов, используемых для лечения бактериальных инфекций. Они действуют, уничтожая бактерии или подавляя их размножение, что позволяет иммунной системе справиться с инфекцией. Антибиотики принимают внутрь (таблетки, капсулы, растворы) или вводят внутривенно. Они неэффективны против вирусов, а неправильное применение ведёт к устойчивости к антибиотикам.
```

Если результата слишком много, можно явно попросить ответ в одно предложение.

Промпт:
```
Антибиотики — это тип лекарственных препаратов, используемых для лечения бактериальных инфекций. Они действуют, уничтожая бактерии или подавляя их размножение, что позволяет иммунной системе справиться с инфекцией. Антибиотики принимают внутрь (таблетки, капсулы, растворы) или вводят внутривенно. Они неэффективны против вирусов, а неправильное применение ведёт к устойчивости к антибиотикам.
Сформулируй вышеописанное в одном предложении:
```

Результат:
```
Антибиотики — это лекарства для лечения бактериальных инфекций, которые убивают бактерии или тормозят их рост; они не действуют на вирусы, а злоупотребление приводит к устойчивости.
```


## 2. 🧲 Извлечение информации

LLM умеют не только генерировать текст, но и извлекать факты из заданного контекста.

Промпт:
```
В авторских вкладках и благодарностях в научных статьях следует чётко указывать, использовали ли авторы ИИ (например, ChatGPT) при подготовке рукописи и анализа, и в какой степени. Также следует указывать, какие модели были использованы. Это позволит редакторам и рецензентам тщательнее проверять материалы на предвзятости, неточности и корректность ссылок. Аналогично, научные журналы должны быть прозрачны в использовании LLM, например, при отборе поданных рукописей.
Назови продукт на основе большой языковой модели, упомянутый выше:
```

Результат:
```
Упомянутый продукт — ChatGPT.
```


## 3. ❓ Вопросно-ответные системы (Q&A)

Структурируйте промпт: добавьте инструкцию, контекст, входные данные и явный индикатор начала ответа.

Промпт:
```
Ответь на вопрос на основе контекста ниже. Дай короткий и точный ответ. Если не уверен — ответь "Не уверен".
Контекст: Препарат теплизумаб берёт начало в компании Ortho Pharmaceutical (Нью-Джерси). Там был создан ранний вариант антитела под названием OKT3. Изначально полученная от мышей молекула могла связываться с T‑клетками и ограничивать их цитотоксичность. В 1986 году её одобрили для профилактики отторжения после пересадки почки — это были первые терапевтические антитела, разрешённые к применению у людей.
Вопрос: Откуда изначально был получен OKT3?
Ответ:
```

Результат:
```
Из мышей.
```


## 4. 🏷️ Классификация текста

Иногда важен не только верный ответ, но и точный формат метки. Покажем несколько вариантов.

Промпт:
```
Классифицируй текст как нейтральная, негативная или позитивная.
Текст: Думаю, еда была нормальной.
Тональность:
```

Результат:
```
Нейтральная
```

Чтобы добиться конкретного формата (например, строчные буквы), добавьте пример:

Промпт:
```
Классифицируй текст как нейтральная, негативная или позитивная.
Текст: Думаю, отпуск был нормальным.
Тональность: нейтральная
Текст: Думаю, еда была нормальной.
Тональность:
```

Результат:
```
нейтральная
```

Покажем проблему с неточной инструкцией:

Промпт:
```
Классифицируй текст как нейтрал, негативная или позитивная.
Текст: Думаю, отпуск был нормальным.
Тональность:
```

Результат:
```
Нейтральная
```

Проблема в том, что вымышленная метка «нейтрал» игнорируется — модель склоняется к знакомой «Нейтральная». Если вам нужна строго «нейтрал», добавьте чёткие определения меток и/или больше примеров.


## 5. 💬 Диалог

С помощью инструкции можно задать манеру речи и роль ассистента.

Промпт:
```
Ниже — диалог с ИИ‑ассистентом для исследований. Тон ответов — технический и научный.
Человек: Привет, кто ты?
ИИ: Приветствую! Я ИИ‑ассистент для исследований. Чем могу помочь?
Человек: Расскажи о возникновении чёрных дыр.
ИИ:
```

Результат:
```
Чёрные дыры — области пространства-времени с настолько сильной гравитацией, что свет не может покинуть их пределы. Они формируются при коллапсе ядра массивной звезды, приводящем к сингулярности с крайне высокой плотностью. Гравитационное поле сингулярности аккрецирует окружающее вещество и излучение, образуя чёрную дыру.
```

Сделаем ответы проще, понятнее для школьников:

Промпт:
```
Ниже — диалог с ИИ‑ассистентом. Отвечай так, чтобы это было понятно даже ученику начальной школы.
Человек: Привет, кто ты?
ИИ: Привет! Я ИИ‑ассистент. Чем могу помочь?
Человек: Расскажи, как появляются чёрные дыры.
ИИ:
```

Результат:
```
Чёрные дыры получаются, когда очень большая звезда «заканчивает своё топливо» и сжимается сама в себя. Её притяжение становится таким сильным, что даже свет не может выбраться — так и получается чёрная дыра.
```


## 6. 🧑‍💻 Генерация кода

LLM хорошо справляются с генерацией кода при чётком описании задачи.

Промпт:
```
/*
Попроси пользователя ввести его имя и выведи "Привет" с этим именем
*/
```

Результат:
```
let name = prompt("Как вас зовут?");
console.log(`Привет, ${name}!`);
```

Более сложный пример c SQL:

Промпт:
```
"""
Таблица departments, колонки = [DepartmentId, DepartmentName]
Таблица students, колонки = [DepartmentId, StudentId, StudentName]
Создай MySQL‑запрос для всех студентов из кафедры "Компьютерные науки"
"""
```

Результат:
```
SELECT StudentId, StudentName
FROM students
WHERE DepartmentId IN (
  SELECT DepartmentId FROM departments WHERE DepartmentName = 'Компьютерные науки'
);
```


## 7. 🧠 Рассуждение

Задачи на рассуждение сложнее. Иногда помогает более точная инструкция и разбиение на шаги.

Промпт:
```
Чему равно 9 000 * 9 000?
```

Результат:
```
81 000 000
```

Более сложный пример:

Промпт:
```
Нечётные числа в этой группе дают в сумме чётное число: 15, 32, 5, 13, 82, 7, 1.
Ответ:
```

Результат:
```
Нет, нечётные числа в этой группе дают нечётную сумму: 119.
```

Ответ неверный. Улучшим инструкцию:

Промпт:
```
Нечётные числа в этой группе дают в сумме чётное число: 15, 32, 5, 13, 82, 7, 1.
Реши по шагам: сначала перечисли нечётные числа, затем посчитай сумму и укажи, чётное это число или нечётное.
```

Результат:
```
Нечётные числа: 15, 5, 13, 7, 1
Сумма: 41
41 — нечётное число.
```

Замечание: даже с улучшенными инструкциями LLM иногда ошибаются. Добавляйте примеры и делайте требования максимально конкретными.
