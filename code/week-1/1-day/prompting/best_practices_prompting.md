# 🔧 Лучшие практики формирования промпт-запросов

Краткое руководство, как писать эффективные запросы к ИИ, чтобы получать точные и полезные ответы.

---

## 🧩 Шаблон хорошего промпта

```
Роль: [кого ты просишь сыграть]
Контекст: [исходные данные, ограничения, цели]
Запрос: [что именно нужно сделать]
Формат ответа: [как именно вернуть результат]
Примеры (опционально): [примеры желаемого результата или ссылки]
Доп. параметры (опционально): [тон, язык, длина, стиль]
```

Пример использования шаблона:
```
Роль: преподаватель по алгоритмам
Контекст: студент готовится к собеседованию, есть 3 дня на подготовку, 2 часа в день
Запрос: составь план подготовки с приоритетами тем и короткими задачами на LeetCode
Формат ответа: таблица по дням + список задач с ссылками
Примеры: нравится подход «от простого к сложному»
Доп. параметры: язык — русский, стиль — лаконичный
```

## 🥗 Пример 1: Составление меню

### Плохой промпт
```
Хочу что-нибудь приготовить.
```

### Хороший промпт
```
Представь, что ты профессиональный домашний повар. Мне нужен рецепт вегетарианского итальянского блюда, которое можно приготовить за 30 минут. Можешь предложить вариант со списком ингредиентов и пошаговой инструкцией?
```

### Разбор
- **Роль:** _"Представь, что ты профессиональный домашний повар"_
- **Контекст:** _"Мне нужно вегетарианское итальянское блюдо на ужин, готовка — не больше 30 минут"_
- **Запрос:** _"Предложи рецепт"_
- **Формат ответа:** _"со списком ингредиентов и пошаговой инструкцией"_
- **Примеры:** Можно указать ссылки на понравившиеся рецепты (например, из `BBC Good Food`), чтобы ИИ лучше понял предпочтения.

## ✈️ Пример 2: Планирование отпуска

### Плохой промпт
```
Хочу в отпуск. Куда поехать?
```

### Хороший промпт
```
Представь, что ты турагент. Нужен план 3-дневного семейного отдыха в Париже с акцентом на детские развлечения. Можешь составить маршрут с расписанием на каждый день и вариантами проживания?
```

### Разбор
- **Роль:** _"Представь, что ты турагент"_
- **Контекст:** _"3-дневный семейный отдых в Париже с акцентом на детские развлечения"_
- **Запрос:** _"Составь маршрут"_
- **Формат ответа:** _"с расписанием на каждый день и вариантами проживания"_
- **Примеры:**
  - Можно показать сайты отелей, которые нравятся
  - Описать места, где уже были, и что понравилось

## 🖥️ Пример 3: Техподдержка

### Плохой промпт
```
Мой компьютер тормозит.
```

### Хороший промпт
```
Представь, что ты специалист техподдержки. Мой MacBook Pro 2019 года на macOS Monterey начинает лагать при открытии нескольких вкладок в браузере. Какие шаги по диагностике и устранению проблемы можно предпринять? Предложи приоритизированный список действий.
```

### Разбор
- **Роль:** _"Представь, что ты специалист техподдержки"_
- **Контекст:** _"MacBook Pro 2019, macOS Monterey, лаги при работе с несколькими вкладками"_
- **Запрос:** _"Какие шаги по диагностике и устранению проблемы?"_
- **Формат ответа:** _"Приоритизированный список действий"_
- **Примеры:**
  - Можно приложить сообщения об ошибках (даже если они выглядят как бессмыслица)
  - Указать, после каких действий появляются проблемы

## 📚 Пример 4: Обучение (например, программированию)

### Плохой промпт
```
Хочу чему-нибудь научиться.
```

### Хороший промпт
```
Представь, что ты преподаватель программирования. Я новичок и хочу освоить Python за 4 недели, занимаясь по 10 часов в неделю. Составь учебный план с ресурсами и недельным расписанием.
```

### Разбор
- **Роль:** _"Преподаватель программирования"_
- **Контекст:** _"Новичок, цель — изучить Python за 4 недели"_
- **Запрос:** _"Составь учебный план"_
- **Формат ответа:** _"С ресурсами и недельным расписанием (10 часов/неделя)"_
- **Примеры:**
  - Можно указать курсы/книги, которые вам нравились раньше
  - Уточнить предпочитаемый формат обучения (видео, книги, практика и т.д.)

---

## ✅ Чеклист перед отправкой промпта
- **Роль** задана и уместна для задачи
- **Контекст** конкретный: цифры, ограничения, приоритеты
- **Запрос** сформулирован как действие/результат, а не тема
- **Формат ответа** указан (список, таблица, шаги, код и т.д.)
- **Примеры** приложены (если есть эталоны)
- **Язык/стиль/тон** уточнены при необходимости
- **Критерии оценки** результата понятны (точность, полнота, краткость)

---

## 🚫 Частые ошибки
- Слишком общий запрос без цели
- Отсутствие контекста и ограничений
- Непонятный формат ожидаемого ответа
- Несогласованность языка и тона
- Игнорирование примеров и эталонов

---

## 🧠 Ключевое
- Конкретика в описании проблемы/цели
- Четкие требования к формату ответа
- Примеры помогают «настроить» ИИ под ваши нужды
