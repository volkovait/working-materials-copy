# Генератор изображений с GigaChat

Проект для генерации изображений с использованием API GigaChat. Позволяет создавать изображения по текстовому описанию и сохранять их локально.

## 🚀 Возможности

- Генерация изображений через GigaChat API
- Автоматическое обнаружение и загрузка сгенерированных изображений
- Автоматическое создание папки `img` для сохранения изображений (на уровне с `src`)
- Уникальные имена файлов с временными метками
- Организованная структура: исходный код отделен от сгенерированных данных
- Обработка ошибок и валидация входных данных
- Подробное логирование процесса с терминально-совместимыми символами

## 📋 Требования

- Node.js версии 16 или выше
- API ключ GigaChat
- Интернет-соединение

## 🛠 Установка

1. Клонируйте репозиторий или скачайте файлы проекта
2. Установите зависимости:
   ```bash
   npm install
   ```

3. Создайте файл `.env` в корне проекта и добавьте ваш API ключ:
   ```
   GIGACHAT_API_KEY=ваш_api_ключ_здесь
   ```

## 🎯 Использование

### Запуск программы

```bash
# Обычный запуск
npm start

# Запуск в режиме разработки (с автоперезагрузкой)
npm run dev
```

### Настройка генерации

По умолчанию программа генерирует изображение котика. Чтобы изменить запрос, отредактируйте файл `demo.js`:

```javascript
content: 'Ваш запрос для генерации изображения',
```

## 🔧 Конфигурация

### Переменные окружения

| Переменная | Описание | Обязательная |
|------------|----------|--------------|
| `GIGACHAT_API_KEY` | API ключ для доступа к GigaChat | Да |

### Настройки клиента

В файле `demo.js` можно изменить следующие параметры:

- `timeout` - таймаут запроса (по умолчанию: 600 мс)
- `model` - модель GigaChat (по умолчанию: 'GigaChat')

## 🚨 Обработка ошибок

Программа включает обработку следующих типов ошибок:

- Отсутствие API ключа
- Проблемы с сетевым соединением
- Пустые ответы от API
- Ошибки при сохранении файлов

## 📊 Логирование

Программа выводит подробную информацию о процессе с терминально-совместимыми символами:

- `[OK]` - Успешные операции
- `[ERROR]` - Ошибки и предупреждения  
- `[INFO]` - Информационные сообщения
- Автоматическое создание папки `img` при первом запуске
- Уникальные имена файлов с временными метками (например: `image_2024-01-15T10-30-45-123Z.jpg`)

## 🔍 Устранение неполадок

### Ошибка "GIGACHAT_API_KEY не найден"
- Убедитесь, что файл `.env` создан в корне проекта
- Проверьте правильность написания переменной
- Убедитесь, что API ключ действителен

### Ошибка "Получен пустой ответ от GigaChat API"
- Проверьте интернет-соединение
- Убедитесь, что API ключ активен
- Попробуйте изменить запрос на более простой

### Ошибка при сохранении изображения
- Проверьте права доступа к папке `img` (на уровне с `src`)
- Убедитесь, что на диске достаточно места
- Проверьте, что файл изображения не заблокирован другим процессом
- Программа автоматически создает папку `img`, если её нет

### Проблемы с отображением символов в терминале
- Убедитесь, что терминал поддерживает UTF-8
- В Windows используйте Windows Terminal или PowerShell
- Символы `[OK]`, `[ERROR]`, `[INFO]` должны отображаться корректно во всех терминалах
