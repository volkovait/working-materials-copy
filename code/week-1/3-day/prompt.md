# Пример промпта для генерации веб-интерфейса

```
Создай статичный лендинг для компании TechNova – лидера в обнаружении и защите от AI-бот атак. Страница должна быть разделена на несколько секций.
Верхняя секция: яркий, современный баннер с логотипом TechNova и слоганом «Защитим ваш цифровой мир от незваных AI-атак». Динамичный фон, с плавными анимациями.
Секция «О нас»: подробное описание компании. Укажи, что TechNova объединяет экспертов в области кибербезопасности и AI-технологий, с опытом работы с крупнейшими мировыми компаниями.
Секция «Ценности»: 
- Надежность и инновации
- Прозрачность и ответственность
- Постоянное совершенствование и обучение
Миссия компании: создание безопасной цифровой среды для бизнеса и пользователей.
Секция «Команда»: краткие биографии ключевых специалистов, международный штат, экспертиза.
Секция «Наши клиенты»: логотипы известных брендов Facebook, YouTube, x.com, google, и других партнеров.
Заключительная секция: форма для обратной связи, кнопка «Запросить консультацию». Используй современный, минималистичный дизайн с акцентом на удобство пользователя.
```
